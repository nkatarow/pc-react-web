<h3>Title Themes:</h3>

```js
<div>
  <Title tag="h1" theme="page-title">Page Title</Title>
  <Title tag="h2" theme="hero-title">Hero Title</Title>
  <Title tag="h3" theme="section-title">Section Title</Title>
  <Title tag="h4" theme="headline">Headline</Title>
  <Title tag="h5" theme="subhead">Subhead</Title>
  <Title tag="h5" theme="subhead caps">Subhead w/ Caps</Title>
  <Title tag="h6" theme="context-head">Context Head</Title>
</div>
```

<h3>Color Examples:</h3>

```js
<div>
  <Title tag="h1" theme="page-title" color="cod-gray">Cod Gray</Title>
  <Title tag="h2" theme="hero-title" color="school-bus-yellow">School Bus Yellow</Title>
  <Title tag="h3" theme="section-title" color="selective-yellow">Selective Yellow</Title>
  <Title tag="h4" theme="headline" color="mai-tai">Mai Tai</Title>
  <Title tag="h5" theme="subhead" color="affair">Affair</Title>
  <Title tag="h5" theme="subhead caps" color="blumine">Blumine</Title>
  <Title tag="h6" theme="context-head" color="faded-jade">Faded Jade</Title>
</div>
```
